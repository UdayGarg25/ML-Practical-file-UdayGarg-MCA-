This lab covers implementation and evaluation of key machine learning algorithms including Decision Trees, Random Forest, SVM, KNN, PCA, and model evaluation techniques. The focus is on comparing performance, tuning parameters, and understanding model behavior.

Experiments Decision Tree vs Random Forest Compare accuracy and performance of both models. Hyperparameter Tuning (Random Forest) Optimize parameters and observe performance changes. SVM (Linear Kernel) Implement and visualize decision boundary on 2D data. Regularization in SVM Analyze effect of parameter C on accuracy and margin. KNN Distance Metrics Compare Euclidean vs Manhattan distance. Feature Scaling (KNN) Study impact of scaling on model performance. PCA Analysis Plot explained variance ratio and select optimal components. Model Evaluation Metrics Use confusion matrix, precision, recall, and F1-score. Cross Validation Compare k-fold cross validation with train-test split. KNN with Different K Values Evaluate performance for varying values of K.

Tools & Libraries Python Scikit-learn NumPy, Pandas Matplotlib / Seaborn

Conclusion Ensemble methods like Bagging/Random Forest improve accuracy. Proper parameter tuning & scaling significantly impact performance. Evaluation metrics help better understand model effectiveness beyond accuracy.
